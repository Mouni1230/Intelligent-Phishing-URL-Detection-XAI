🛡️ Phishing URL Guard – README

📌 Project Overview

Phishing URL Guard is a simple yet effective web-based tool designed to help users detect potentially malicious or phishing URLs.
The system analyzes the given URL using basic keyword-based filtering and displays whether the link is safe or unsafe.
It also includes modern UI design, animated backgrounds, and a smooth scanner-style loading animation.

This project demonstrates the frontend implementation of a phishing detection interface and can be extended with machine learning or backend APIs.

🔍 How the Detection Works

The JavaScript logic checks for suspicious keywords in the URL:

"login"

"verify"

"secure"

"update"

"bank"

"free"

"click"

If any of these appear in the URL, the system marks it as ⚠️ Unsafe / Possible Phishing.
Otherwise, it shows ✔ Safe.

> This is a basic simulation.
> The system can later be upgraded with:

Machine learning

API-based threat analysis

Domain reputation scoring

WHOIS lookup
