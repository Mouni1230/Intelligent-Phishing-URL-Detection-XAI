# 🛡️ Phisentia - Intelligent Phishing URL Detection using Machine Learning and Explainable AI

## 📖 Overview

Phisentia is an AI-powered phishing URL detection system that detects malicious websites using Machine Learning and Explainable AI (XAI). The application predicts whether a URL is **Legitimate** or **Phishing** and provides explanations using **SHAP** and **LIME**, helping users understand why a prediction was made.

---

## 🚀 Features

- 🔍 Real-time phishing URL detection
- 🌐 Chrome Browser Extension
- 🤖 Decision Tree Machine Learning model
- 📊 Confidence score for predictions
- 💡 Explainable AI using SHAP and LIME
- ⚡ Flask REST API
- 🛡️ Detects malicious websites before users interact with them

---

## 🌐 Chrome Extension Setup

1. Open Google Chrome.
2. Go to `chrome://extensions/`.
3. Enable **Developer Mode**.
4. Click **Load unpacked**.
5. Select the `phishing_extension` folder.
6. Start browsing and detect phishing websites in real time.

## 📊 Explainable AI

### SHAP
SHAP explains the contribution of each feature to the model's prediction, improving transparency and trust.

### LIME
LIME explains individual predictions by highlighting the most influential features for a specific URL classification.
